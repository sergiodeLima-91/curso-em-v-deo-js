Document Object Model

Conceito

Modelo de objetos para documentos. Conjunto de objetos dentro do browser que conferem acesso aos componentes internos de seu website. Não funciona dentro do NODE, mas sim dentro do navegador.

Árvore DOM (window)

Dentro dele existem diversos outros objetos. São alguns deles:

  1. location: localização do site (URL)
  2. document: documento atual. Dentro do document temos o HTML com dois childs (head e body)
  3. history: histórico de navegação (?)

Navegando no DOM por Seleção de Elementos da Árvore:

Podemos alterar e manipular os elementos (tags) do documento HTML através de seus elementos. Quais as formas de navegação?

  1. Por MARCA - getElementsByTagName()
  2. Por ID - getElementById()
  3. Por Nome - getElementsByName()
  4. Por Classe - getElementByClassName()
  5. Por MARCA - getElementByTagName()

Perceba que no segundo exemplo a palavra objeto veio em negrito, exatamente como esta no segundo parágrafo. Isso porque INNER TEXT pega o texto sem as tags filhas e formatações,mas o INNER HTML pega todo o texto HTML.
